# Simple Calendar


#### Quick Start

```shell
$ git clone git@gitlab.com:azizka/cala-calendar.git
$ cd cala-calendar
$ yarn
$ yarn start
```


#### Yarn/NPM Commands

|Script|Description|
|---|---|
|`yarn start`|Start the app on `localhost:3000`|
|`yarn build`|Build the app to `./build` directory|
